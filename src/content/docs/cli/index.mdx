This tutorial variant is reserved for the command line interface.
It is useful once you are comfortable with the basic Git ideas and want a workflow that is independent of any specific website.

At the moment, the command line section is only scaffolded.
The selector in the top-left corner already supports it, so CLI-specific guides can be added step by step without changing the site structure again.

<Aside type="note" title="Why keep CLI as a separate variant?">
  GitHub, GitLab, and the command line all work with the same underlying Git concepts,
  but the actions look very different.
  Keeping them as separate variants avoids mixing screenshots, button names,
  and terminal commands on the same page.
</Aside>
