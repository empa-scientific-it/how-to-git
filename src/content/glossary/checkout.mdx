---
term: Checkout
definition: Switching from one branch or commit to another.
related: [branch, commit]
---

**Checking out** changes your working directory so you can work on a different branch or view an older version of the project. When you checkout:

- Git updates your files to match the selected branch or commit
- Your previous work remains safely stored in its branch
- You can move between branches whenever needed
- Modern Git often uses git switch when changing branches

Older tutorials commonly use `git checkout`, while newer ones often recommend `git switch`.
